Expand description

Renders the preview panel for the currently selected entry.

  • Directories: shows an indented tree snapshot (depth 3, max 50 entries).
  • Text files: shows syntax-highlighted content with line numbers.
  • Binary files: shows a size message.
  • No selection: shows a placeholder.
